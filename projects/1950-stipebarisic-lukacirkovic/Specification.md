# Project Specification: PC Builder

## Overview
 This is a web application for Build a PC with your available budget.
 
## Functional Requirements
- Search PCs
- Review of the best and most affordable websites that sell components
- Favorites

## Non-Functional Requirements
- Best Performance
- Best Option
- Fast browser and time saver
